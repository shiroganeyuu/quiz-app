# Quiz Frontend

React + TypeScript + Tailwind CSS application for the Quiz App.

## Getting Started

### Installation

```bash
cd quiz-frontend
yarn install
```

### Development

```bash
yarn dev
```

The app will open at `http://localhost:3000`

### Build

```bash
yarn build
```

### Preview

```bash
yarn preview
```

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
