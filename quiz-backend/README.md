# Quiz Backend

Node.js + TypeScript backend API for the Quiz App.

## Getting Started

### Installation

```bash
cd quiz-backend
yarn install
```

### Development

```bash
yarn dev
```

The server will run on `http://localhost:5000`

### Build

```bash
yarn build
```

### Start (Production)

```bash
yarn start
```

## Tech Stack

- Node.js
- Express.js
- TypeScript
- CORS support

## API Endpoints

- `GET /api/health` - Health check
- `GET /api/quizzes` - Get list of quizzes
